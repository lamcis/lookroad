 **看路 (LookRoad)** 是一款 Android 全局透明相机悬浮窗辅助工具。它能在手机屏幕上层显示后置摄像头的实时画面，并实现**完全的触摸穿透**（不影响其他 App 正常操作）。让用户在户外必须使用手机的场景下，能透过屏幕察觉前方环境。
 LookRoad is an Android tool that provides a globally transparent camera floating window. It displays the live feed from the rear camera on top of the phone screen, allowing for complete touch penetration (without affecting the normal operation of other apps). This enables users to perceive their surroundings through the screen when they must use their phone outdoors.

 **特性**：
-  **触摸穿透**：无感运行，不阻挡任何点击与滑动。
-  **边缘悬浮球**：支持拖拽与自动吸附，快捷调节透明度、屏幕亮度及手电筒。
-  **安全纯净**：纯本地运行，零隐私收集。
